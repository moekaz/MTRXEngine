A basic C++ based physics engine which uses some libraries like G\+LM, S\+P\+D\+L\+OG, G\+L\+AD, G\+L\+FW.


\begin{DoxyItemize}
\item Implements rigidbody dynamics with force and torque integration.
\item Implements force generators that can be used to simulate certain forces like gravity, drag, buoyancy, stiff springs, etc...
\item Implements a basic collision detection system with basic bounding volumes (sphere, capsule, box).
\item Implements convex shape colliders with G\+JK Collision detection
\item Basic raycasting that can be used to grab a certain collider using a certain ray with the option to filter out certain colliders
\item Very rough implementation of Bounding Volume Hierarchies(untested) that can be used to optimize collision detection checks
